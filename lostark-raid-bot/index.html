What it does

  • Raid scheduling with Discord slash commands — create, join, and manage raid groups without leaving Discord
  • Roster management: track signups, set role slots, and handle waitlists automatically
  • Automatic reminders before scheduled raids
  • Versioned commands with self-healing error recovery and diagnostic commands for operators
  • Backup, restore, and migration operations for raid data

Status

The bot is live and in active use. It is a private deployment — source is not public at this time.
