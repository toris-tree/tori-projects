# Setup Guide

This is the short markdown setup flow. For the full visual guide with screenshots, use:

https://lostark.cepcar.top/setup-guide.html

## Quick Setup

1. Add Lostark Raid Bot to your Discord server.
2. In Discord, type `/help`.
3. Start the private guided setup with `/toonsetup`.
4. Choose how many characters you want to add.
5. Enter character names.
6. Enter item levels in the same order as the names.
7. After the roster saves, type `/raids` to see eligible raids.
8. Later, use `/raidsleft` to check unfinished weekly raids.

## Useful Follow-Up Commands

- `/toons` - view your saved characters
- `/toonadd` - add one more character
- `/toonedit` - fix a character name or item level
- `/toondel` - remove a character
- `/theme` - change dashboard color style
- `/help` - show the quickstart again

Most setup replies are private. Do not enter passwords, account login details, bot tokens, or other sensitive information into Discord bot forms.
