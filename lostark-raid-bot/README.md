# Lostark Raid Bot

Lostark Raid Bot is a Discord bot for Lost Ark roster setup, raid eligibility, weekly raid tracking, and quick raid-status checks inside Discord.

## Install

Add the bot to a Discord server:

[Install Lostark Raid Bot](https://discord.com/oauth2/authorize?client_id=1476709196325191810&scope=bot%20applications.commands&permissions=277025508352)

The bot uses Discord slash commands and does not require Administrator permission.

## Basic Commands

- `/help` - show the quickstart and support links
- `/toonsetup` - guided private roster setup
- `/toons` - view your saved characters
- `/toonadd` - add one character
- `/toonedit` - fix a character name or item level
- `/toondel` - remove a character
- `/theme` - change dashboard color style
- `/raids` - open your raid dashboard
- `/raidsleft` - see unfinished weekly raids
