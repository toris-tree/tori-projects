# Security Policy

## Do Not Share Secrets Publicly

Never post any of these in GitHub issues, Discord support messages, screenshots, or public comments:

- Discord bot tokens
- private Discord user tokens
- passwords
- account credentials
- API keys
- session cookies
- private account or payment information

If you accidentally share a secret publicly, rotate or revoke it immediately.

## Reporting Sensitive Issues

For public bugs and feature requests, use GitHub issues.

For sensitive issues, do not open a public issue. Contact support through:

- Support Discord: https://discord.gg/7zppGQzF8C
- Email: 4cepgaming@gmail.com

Keep the first report high-level. Do not include secrets unless a private, trusted channel has been established and the maintainer specifically asks for the minimum needed detail.

## Supported Project

Lostark Raid Bot is in public preview. Security and privacy reports for the public bot, website, setup flow, and support process are welcome.
